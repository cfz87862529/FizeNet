# FizeNet
网络相关操作类。

FizeNet 是对一些常用的系统内置的网络函数(如 curl )进行面向对象的再封装类库，方便调用。 

FizeNet 还提供了一系列的网络类来简化网络操作。

FizeNet 是 Fize 项目族群的底层依赖，众多 Fize 项目( 特别是第三方功能 )依赖 FizeNet ，同时欢迎您使用 Fize。 

FizeNet 有非常完善的参考文档，其代码结构也经过精雕细琢，精炼而实用。
